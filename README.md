# Modal CSC434 Projet - Deep Learning in Computer Vision

The project takes the form of a Kaggle Competition on the Something-2-Something dataset, which is a video labeling competition of various actions performed in front of a camera.

## Quick Start

### Environment

Install requirements and set up virtual environment using uv.

```bash
uv venv --python 3.12.9
```

```bash
uv sync
```

### Obtain Data

Run

```bash
uv run scripts/download_data.py
```
